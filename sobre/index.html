- Quais liguagens você já teve pelo menos algum contato?

Para Web eu tive contato apenas com PHP(puro) e as ferramentas front-end convencionais (HTML, Javascript e CSS). Além disso também vi C e C++, e gostei bastante, mesmo com a dor de cabeça de não obter as coisas de forma simples - mas o legal de programar é isso, rs. Também cheguei a ter contato com o SGBD MySql.

Com toda certeza C++, apesar de gostar de python eu não gosto de sentir a limitação de entender como a linguagem é abstraida para chegar a cada solução. Atualmente quero aprender o framework QT para C++, mas em paralelo quero aprender tecnologias web mais novas para me adaptar às tecnologias mais visadas na atualidade.
